# 代码高亮示例代码
本项目是使用React + prismjs实现代码高亮示例。

# 使用
项目需要先全局安装依赖库[umi](https://github.com/umijs/umi)。

```
npm install -g umi
```

然后Clone项目在运行
```
git clone https://github.com/scfido/codeHighlighter
cd codeHighlighter
npm install
npm start
```
